# Sales Performance Dashboard & Data Analyzer

An Excel-based data analytics project built using advanced Excel features for sales analysis, KPI tracking, forecasting, and dashboard visualization.

## Features
- Interactive Sales Dashboard
- KPI Tracking & Profit Analysis
- Regression & Statistical Analysis
- Pivot Tables & Trend Analysis
- Customer Segmentation
- What-If Analysis using Data Tables
- Automated Insights with Excel Functions

## Sheets Included
- Dashboard
- Dataset
- Task_1 to Task_9

## Tools & Concepts Used
- Pivot Tables & Charts
- Data Tables (What-If Analysis)
- Regression Analysis
- Conditional Formatting
- Excel Formulas & Functions
- KPI Reporting

## Project Goal
To analyze sales data and generate meaningful business insights using Microsoft Excel.
